# Jacky Zhang — Personal Website

## What This Is

A personal website for Jacky Zhang — a Computer Engineering student at UIUC and a multi-domain builder (Roblox games with 30k+ plays, founding developer at a stealth fintech startup, Team Canada RoboCup robotics). The site is a recruiting-focused portfolio that signals "I ship real things" to tech recruiters and engineering hiring managers, leading with a bold, founder-energy visual identity.

## Core Value

A recruiter or hiring manager lands on the site and within seconds understands that Jacky is a serious builder worth interviewing — backed by concrete projects and credibility signals.

## Constraints

- **Tech stack**: Next.js + React + TypeScript + Tailwind CSS + Framer Motion, deployed on Vercel — modern industry-standard stack that doubles as a recruiting signal and overlaps with known React skills.
- **Design direction**: Bold & modern "founder" aesthetic, dark-first — locked design system (see Key Decisions).
- **Scope**: v1 is a focused, static, recruiting-oriented portfolio. No blog, CMS, or backend.
